# App Icons Directory

Place your app icons here in the following sizes:

## Required Sizes:
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## How to Generate:

Use one of these online tools:
1. **Icon Kitchen**: https://icon.kitchen/ (Recommended)
2. **PWABuilder**: https://www.pwabuilder.com/imageGenerator

Or see the full guide: `../../ICON_GENERATION_GUIDE.md`

## Quick Start:

1. Create a 1024x1024 px logo/icon
2. Upload to Icon Kitchen
3. Download the ZIP
4. Extract PNG files to this directory
5. Done! Your PWA will use these icons.
